# Frontend (Vite + React)

Requirements:
- Node.js

Quick start:
1. Install dependencies: `npm install` inside `frontend`.
2. Start dev server: `npm run dev`.

Notes:
- App expects backend at `http://localhost:5000`.
- Main files: `src/pages/Dashboard.jsx`, `src/components/*`.
- Uses `axios` for API calls and `react-chartjs-2` for charts.
